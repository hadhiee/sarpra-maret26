# Rangkuman Bidang IT, Laboratorium, dan Sarpras

Dokumen ini merangkum materi rapat bulanan Maret 2026 untuk SMK Telkom Malang, dengan fokus pada penguatan ekosistem digital, pengembangan laboratorium, serta pembaruan sarana dan prasarana.

## Isi Halaman

Halaman utama ada di file index.html dan memuat bagian berikut:

1. Bidang IT
- Arah transisi platform ke Microsoft 365.
- Peralihan meeting dari Zoom ke Microsoft Teams.
- Penguatan fasilitas IT untuk pembelajaran berbasis proyek.
- Awareness perlindungan data pribadi (PDP).

2. Bidang Laboratorium
- Pengembangan wadah entrepreneur siswa, khususnya non-IT.
- Dukungan terhadap program BMW (Bekerja, Melanjutkan, Wirausaha).
- Pencapaian HKI program komputer: MoLeCuL, Mexpo, myedotel.id, Telkom Society, Portofolio Siswa, dan Moklet App.

3. Bidang Sarana Prasarana
- Usulan kebutuhan CAPEX 2026, termasuk MTP, perbaikan ruang, atap, dan pengembangan area pendukung Teaching Factory.
- Status pengajuan: dalam evaluasi dan tindak lanjut prioritas.

4. Informasi Utilitas Gedung
- Monitoring penggunaan listrik dan air.
- Informasi operasional: 3 panel listrik utama, estimasi tagihan listrik Maret 2026, serta pemantauan PDAM.

5. Penutup
- Komitmen bersama tim IT, Laboratorium, dan Sarpras untuk mendukung pembelajaran yang aman, inovatif, dan berkelanjutan.

## Tujuan Dokumen

- Menjadi ringkasan cepat untuk kebutuhan rapat bulanan.
- Menyediakan narasi yang mudah dipahami oleh guru dan tim manajemen.
- Menegaskan prioritas pengembangan sekolah pada aspek digital, inovasi, dan fasilitas.

## Cara Mengakses

1. Buka file index.html di browser.
2. Gunakan menu navigasi cepat pada bagian atas halaman untuk berpindah antarbagian.

## Catatan

Konten disusun untuk kebutuhan internal rapat bulanan SMK Telkom Malang, Maret 2026.
