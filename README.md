# Rangkuman Bidang IT, Laboratorium, dan Sarpras

Dokumen ini merangkum materi rapat bulanan Maret 2026 untuk SMK Telkom Malang, dengan fokus pada penguatan ekosistem digital, pengembangan laboratorium, perlindungan karya inovasi, serta pembaruan sarana dan prasarana.

## Isi Halaman

Halaman utama (`index.html`) telah disusun ulang untuk memprioritaskan profil unit dan personil, dengan rincian sebagai berikut:

1. **Proses Bisnis Unit Sarpras**
   - Diagram alur kerja terintegrasi untuk unit Sarpras, IT, dan Laboratorium.
   - Peningkatan awareness alur kerja untuk efisiensi layanan sekolah.

2. **Personil Unit IT, Lab, dan Sarpras**
   - Daftar personil dan struktur tim pengelola bidang IT, Lab, dan Sarpras.

3. **Bidang IT**
   - Arah transisi platform ke **Microsoft 365**.
   - Peralihan platform meeting secara bertahap ke **Microsoft Teams**.
   - Penguatan fasilitas software dan alat praktik.
   - Awareness **Perlindungan Data Pribadi (PDP)** bagi seluruh unit kerja.

4. **Bidang Laboratorium**
   - Pengembangan wadah entrepreneur siswa (BMW: Bekerja, Melanjutkan, Wirausaha).
   - Capaian HKI Program Komputer: MoLeCuL, Mexpo, myedotel.id, Telkom Society, Portofolio Siswa, Moklet App.
   - **Sertifikat HKI Moklet**: Kumpulan sertifikat pencatatan hak cipta yang telah diraih.

5. **Bidang Sarana Prasarana**
   - Usulan kebutuhan **CAPEX 2026**.
   - Fokus: Pengembangan Gedung **MTP (Malang Techno Park)**, perbaikan infrastruktur gedung, dan ruang praktikum.
   - Model 3D interaktif untuk visualisasi rencana Gedung MTP.

6. **Informasi Utilitas Gedung**
   - Monitoring operasional 3 panel listrik utama dan PDAM.
   - **Perbandingan Tagihan (4 Bulan)**: Rekapitulasi biaya dari Desember 2025 hingga Maret 2026.
   - Grafik tren biaya untuk budaya efisiensi energi.

7. **Penutup**
   - Komitmen tindak lanjut kolaboratif 30-60-90 hari.

## Fitur Tambahan

- **Mode Presentasi**: Alur slide yang selaras dengan halaman utama untuk kebutuhan pemaparan rapat.
- **Visualisasi Dinamis**: Menggunakan Chart.js untuk grafik utilitas dan Tripo3D untuk model infrastruktur.

## Cara Mengakses

1. Buka file `index.html` di browser.
2. Gunakan menu navigasi cepat pada bagian atas halaman untuk berpindah antarbagian.
3. Klik tombol navigasi (atau tekan keyboard) untuk beralih ke Mode Presentasi.
